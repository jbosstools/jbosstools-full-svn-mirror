New name for Seam Tools

We renamed Seam UI's such as Seam Perspective, New Seam Project Wizard, etc., to Seam 2 in order not to confuse users who are looking for Seam 3 Tooling. JBoss Tools also support Seam 3 but it's a separate tooling included in CDI Tools.
