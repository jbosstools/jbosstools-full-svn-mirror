Deltacloud Java Client
Moved to Apache Foundation We developed a java REST client at JBoss Tools, that allowed our Eclipse based tooling to communicate with the Deltacloud server.
We now contributed this java client to the Deltacloud project at the apache foundation. It will now get maintained by the Deltacloud project.
