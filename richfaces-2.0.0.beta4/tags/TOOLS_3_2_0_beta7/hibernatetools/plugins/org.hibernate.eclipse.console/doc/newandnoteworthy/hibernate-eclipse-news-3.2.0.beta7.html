Release notes

Reverse engineering

Many-to-Many

By popular demand the reverse engineering now supports automatic detction of many-to-many. It is enabled by default, but can be toggled in the UI or in Ant via the detectmanytomany attribute.

Optimistic locking columns

Detection of optimistic locking columns was enabled in the previous versions and customizable programmatically, but now it is also available in the UI and Ant task (detectoptimisticlock attribute).

Query editing

Null in parameters

The query parameter view now has an additonal column that can be enabled/disabled dependent on you want null to be set instead of a concrete value.
