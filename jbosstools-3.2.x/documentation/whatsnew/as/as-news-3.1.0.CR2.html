General

"Full publish" keyboard shortcut

There are now a keyboard shortcut for "Full publish" allowing you to publish the currently selected project without using mouse and clicking on the Servers View.

The default keybinding is Alt+Shift+Y F and its now sharing the same base (Alt+Shift+Y) with "Build project archives" which now is (Alt+Shift+Y B).

JMX Poller now timeouts by default

If the JMX "ping" to check if a server is fully started now no longer makes the server looks like it never started. We will now timeout silently allowing servers that does not have JMX configured (such as the "Minimal" configuration) to be viewed as Started.
