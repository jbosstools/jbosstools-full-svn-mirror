Eclipse 3.6 (Helios)

All JBoss Tools now targets Eclipse 3.6; this allows us to use the new API's and provide better integrated user experience within Eclipse. Some plugins will possibly still run on previous Eclipse versions but we cannot guarantee it. If you want to stay on Eclipse 3.5 then use JBoss Tools 3.1.
