CDI Support enablement

CDI Support Settings Page

You can enable or disable CDI support for a non-faceted project via Configure...->Add CDI Support or Project Settings page (Project->Properties->CDI).



Related Jira

CDI Web Project Facet

It's possible to enable CDI support for a faceted project via New Dynamic Web Project wizard or Project Facets settings page (Project->Properties->Project Facets).

Related Jira

Project Validation

EL Validation

EL validator now is aware of CDI @Named beans for CDI and JSF enabled projects.

Related Jira

Core Validation

As part of the CDI tooling validation of certain CDI rules are now validated during project build. You can configure the severity of the CDI Validation rules on the CDI Preference page.



Related Jira

Code Completion

EL Code Completion

EL Code Completion now works for @Named beans and it uses the Weld icon to allow you to see they are CDI beans.
