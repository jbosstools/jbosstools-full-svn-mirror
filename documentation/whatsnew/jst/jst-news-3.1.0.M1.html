OpenOn for id referencing attributes

We now support navigating from i.e. a4j:commandButton focus="customer" to the tag with the id "customer".

This OpenOn works for a long range of tags and attributes, thus if you have an attribute that refer to an id it should now work. If not, let us know so we can extend the support.
