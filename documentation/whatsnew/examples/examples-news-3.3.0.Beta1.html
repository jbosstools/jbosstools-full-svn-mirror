Project Examples
New Project Example wizard

JBoss Tools now includes a new wizard to download and import project examples.
The wizard starts using Help>Project Examples or by clicking an item in the JBoss Central editor.

Related Jira
Project Examples can be dynamically added to JBoss Central

A project example can be dynamically added to the Project Examples section of the JBoss Central editor.
It is enough to add the central tag to the project example xml file. The JBoss Quickstarts category in http://download.jboss.org/jbosstools/examples/project-examples-maven-3.3.xml can serve as an example.

Related Jira
Defining a category

Now it is possible to add a description and priority for categories as well as a priority for project examples. See http://download.jboss.org/jbosstools/examples/project-examples-maven-3.3.xml
