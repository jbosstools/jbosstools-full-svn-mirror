< Main Index Smooks Tools News >

ESB
BPMN to jPDL Export Wizard

We added an export wizard for translating BPMN files created by Eclipse STP BPMN editor to jPDL that can be executed with the JBoss jBPM process engine.

The wizard can be activated from the File > Export menu.
ESB UI improvement

There is now a unique icon for ESB related files/wizards.
ESB Project Deployment

There is now a ESB project Wizard which creates a project that will be deployed as an .esb archive to a JBoss AS based server.
jboss-esb.xml editor

We added an structured xml editor for the jboss-esb.xml file used in JBoss ESB archives.
