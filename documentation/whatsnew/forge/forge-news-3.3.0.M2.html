Name Change

Not a lot of new work has been done in the Forge Tools area for M2. The most important thing is a change of the names of the plugin, feature and Java packages to remove the 'seam' reference. This means that if you upgrade from M1 the normal update mechanism will not work as expected for the Forge Tools. Two possible workarounds are to either manually delete the old forge plugin and feature (that is the one that has 'seam' in the name) or else to just create a complete new JBoss Tools installation based on 3.3.0.M2.
