Conditional Transitions

Conditional transitions are discouraged if they are leaving other nodes than condition nodes. The property page for the conditions on transitions will display a warning if a condition is added to such a transition. This is illustrated in the below screenshot.

The warnings can be suppressed by checking the checkbox on the new 'Compatibility' preference page.
