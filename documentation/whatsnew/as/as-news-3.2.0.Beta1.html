Server

Remote System Support Improvements

Remote system support has been improved. Before and after publishes, the remote system will now suspend its deployment scanner to make sure that the full deployment is published before the server tries to register it. Also, a page to customize start and stop commands has been added to the launch configuration.

Web Startup / Shutdown Poller added

A new poller has been added. This poller scans the web port for a response. It is a simpler poller which does not require JMX to be installed and running, and is now the default poller for all newly-created servers.
