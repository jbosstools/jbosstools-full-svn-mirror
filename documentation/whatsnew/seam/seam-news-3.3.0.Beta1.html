Seam 2.3

JBoss Tools now support Seam 2.3. It includes Seam wizards, Seam project facet, runtime detection, maven integration, components.xml file editor and all the other features which are available for Seam 1 - 2.2.

@SuppressWarnings

Seam Validator now supports @SuppressWarnings annotation. There is a quick fix available for every validation warning which adds the corresponding @SuppressWarnings. You can find the full list of all the available warning names in Jira.
