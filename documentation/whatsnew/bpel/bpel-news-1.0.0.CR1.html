General
BPEL Editor A BPEL Editor based on the Eclipse BPEL project has been added to JBoss Tools. This means you can now create, edit and deploy BPEL artifacts for the Riftsaw BPEL Runtime.

Wizards Added a 'New' Wizard to create a BPEL project in Eclipse which creates WTP compatible .bpel project that can be deployed to a BPEL runtime such as Riftsaw or Apache ODE. See the BPEL tools documentation for details.
