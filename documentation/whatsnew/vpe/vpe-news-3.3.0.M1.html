General
XULRunner 1.9

Visual Editor is now based on Mozilla Xulrunner 1.9.2.16 - the same rendering engine on which Firefox 3.6.16 is built.

This means support for new CSS attributes, better performance and many improvements for web developers, including HTML5 support which will be added to the Editor in the near future.
