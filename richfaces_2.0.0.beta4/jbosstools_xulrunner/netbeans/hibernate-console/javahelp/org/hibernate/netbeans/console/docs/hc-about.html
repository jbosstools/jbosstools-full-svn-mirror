The Hibernate Console

The Hibernate Console enables you to test/prototype HQL queries. It consists of the following parts:

Session Factories Window
displays all managed session factories. Accessible from the menu entry Window | Hibernate Console | Session Factories
HQL Editor
editor for HQL queries. It is automatically opened when the session factory is opened
BeanShell Editor
editor for custom java-code. It is automatically opened when the session factory is opened
Query Results Window
displays query results. It is automatically opened when the session factory is opened or when a query is executed
SQL Logging Window
shows all SQL statements executed by hibernate. Accessible from the menu entry Window | Hibernate Cosole | SQL Logging
