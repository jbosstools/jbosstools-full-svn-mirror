The Hibernate Console

The Hibernate Console enables you to test/prototype HQL queries. It consists of the following parts:

Session Factories Window
displays all managed session factories. Accessible from the menu entry Window | (HC) | Session Factories
HQL Editor
editor for HQL queries. Accessible from the menu entry Window | (HC) | Query Editor
Query Results Window
displays query results. Accessible from the menu entry Window | (HC) | Query Results
SQL Logging Window
shows all SQL statements executed by hibernate. Accessible from the menu entry Window | SQL Logging
