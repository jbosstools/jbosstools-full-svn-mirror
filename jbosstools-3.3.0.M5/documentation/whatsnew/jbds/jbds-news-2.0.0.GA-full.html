JBoss Developer Studio 2.0.0.CR1 News

JBoss Developer Studio 2.0.0.CR1 - New and Noteworthy

Installer

Installer without runtime

We now provide an installer without a JBoss EAP runtime for users with an existing EAP runtime and users who like to save bandwidth.

Additional servers

The installer now allow users to setup additional servers which will be automatically configured for any new workspaces.

The installer will try and detect what type of server the user have pointed to (EAP, SOA-P, AS, etc.) and configured the apropriate runtimes. i.e. for SOA-P the server, seam and jbpm runtimes are configured where as if you only point to the AS it will only setup the server.
