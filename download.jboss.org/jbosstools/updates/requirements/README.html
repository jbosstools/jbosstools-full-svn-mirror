JBoss Tools Build Requirements - Update Sites

This directory contains a number of child folders, each containing update sites (p2 repos) of third party plugins and features used to build JBoss Tools or contained in JBoss Developer Studio.

While you can certainly install these plugins from the individual URLs of the folders nested below here, it is by far easier to simply add the composite sites for your version of Eclipse / Web Tools / JBoss Tools / JBoss Developer Studio, and install from that single URL. The big reason for this - aside from the fact that 1 URL is easier than several, is that the composite site also includes all the third party projects' upstream dependencies (eg., Eclipse, WTP, Orbit), making it a one-stop shopping destination.

  • All JBoss Tools 3.3 dependencies in one update site (for installing or developing with Eclipse 3.7 / Web Tools 3.3 / Indigo)
  • All JBoss Tools 3.2 dependencies in one update site (for installing or developing with Eclipse 3.6 / Web Tools 3.2 / Helios)

If you want a restricted set of features/plugins, namely the target platform used to build JBoss Tools or JBoss Developer Studio, click here.
