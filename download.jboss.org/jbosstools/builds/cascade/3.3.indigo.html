Build Job Cascade:

  • Local Target Platform build (build + publish)
    • Continuous build (build only, no tests, no publish)
    • Tests build (build + tests, no publish)
    • All components' jobs, including JBT - Core, JBT - SOA Tooling, Drools, Teiid, pi4soa, Savara (build, test, publish) :: list of unstable jobs - Interdependencies between jobs, aka "swimlanes"
      • JBT - Web Tools build (collect components into a single update site for use as WTP Server Adapter)
        • JBT - Core build (collect components into a single update site for downstream use and publishing to sf.net)
          • JBT - SOA Tooling build (collect components into a single update site for downstream use and publishing to sf.net)
      • JBT - Core Tests build (collect all tests into a single update site)
        • JBT - SOA Tests build (collect all tests into a single update site)
  • Local JBDS Target Platform build (build + publish)
      • JBDS Aggregate site build (build Core Subset [JBT + JBDS], Extras, Tech Preview sites, then publish)
        • JBDS Product builds (combine JBDS components and TP into Core Site + Installers, publish) DEPRECATED after 5.0.0.GA
        • JBDS SOA Tooling site build (build site, then publish)
